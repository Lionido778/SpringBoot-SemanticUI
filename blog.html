javascript 笔记


一.Javascript简介

（一）.Javascript 能用来干什么？

  • 数据的验证。
  • 将动态的文本写入到网页中。
  • 可以对事件做出响应
  • 可以读写HTML中的内容。
  • 可以检测浏览器。
  • 可以创建 cookies 。

（二）.Javascript特点

基于对象和时间驱动的松散型的解释性语言
  1. Javascript面向对象开发的
  2. 事件驱动
  3. 松散型 弱类型 var a="asc";
  4. var a=123;
  5. 解释性语言

（三）.历史

  • JScript
  • Scriptease
  • JAVASCTIPT
  • ECMAscript

（四）.Javascript 的构成

  • ECMAscript(语法)
  • BOM browser ObJECT Model
  • DOM document Object Model

(五).如何入门

  1. html css xml
  2. 开发工具

二.Javascript 的放置和注释

(一.). 输出工具

  1. alert()
  2. document.write
  3. prompt

(二).Javascript 如何在 html 页面当中进行放置

  • [Javascript 可以在 html 页面中的任何位置来进行调用，但是它们还是一个 繁体 是相互联系 相互影响。]
  • 可以在超链接或是重点向的位置调用 Javascript 代码 格式："javascript:alert(我是超链接）" 中定向定格： "action=javascript( 我是表单 ）"

(三).在事件后面进行调用

  • 格式：οnclick="alert(我是事件)"

(四).调用外部 Javascript 文件

  • 格式： [在调用页面 script 标签对当中不能有任何代码，在JS 脚本中不能出现 script 标签对 但是它们还是一个繁体 是相互联系，互相影响。]

(五).Javascript 注释：

  1. 对付旧的浏览器
    2. 真正的注释符号:
    • 行内注释 //
    • 块注释 /* * /

(一).Javascript 命名规范

  • 严格区分大小写
  • 变量的命名必须以字母或 _ $ 开头，余下的部分可以使任意的字母 数字 或者是 _ 或者是 $
  • 不能用关键字或者是保留字命名。
    • <
  • Javascript 自己的命名习惯 驼峰命名法：getElementByld 首字母大写：Object
  • 命名一定要有意义。
  • 的用法

(二).Javascript 变量

变量：可以储存数据的一个容器。
  • 变量如何创建（声明）必须以“var”关键来修饰。
  • 先声明，后赋值 var bbs; bbs="bbs.houdunwang.com";
  • 声明和赋值同时进行 var url="www.houdunwang.com";
  • 一次声明多个变量，然后再赋值 var name.age,sex; name="lisi"; age=14; sex="boy"
  • 一次声明多个变量同时进行赋值。 var name ="wangwu".age=17.sex="girl";
  • 如何覆盖已有变量
  • 如何重新声明该变量，而没有赋值，该变量的值不会改变
  • 如果重新声明该变量并且重新赋值，那么旧变量值会删除， 改为新的变量值。
  • 不用 var 关键字来修饰变量 aa="我是没有声明的" alert(aa) 如果不用 var 关键来修饰的变量，并且这个所谓的变量也没有赋值 那么 Javascript 会报错。 如果不用 var 关键来修饰的变量，但是变量赋值了。那么 Javascript 会把它当做一个全局变量来处理 不会报错。但是我们不推荐使用。

    • Number

    • Boolean
  2. 引用类型
    • object (class)
